# PdfImg

A React component to render pdf documents as images.


## Installation

Install it as any other `npm` module:

```bash
$ npm install pdf-img
```


## Usage

```javascript
  <PdfImg
    src={testPdf}
    page="1"
    style={{ width: 300 }} />
```
